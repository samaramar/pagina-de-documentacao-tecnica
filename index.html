Introdução

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. Foi criada por Guido van Rossum e lançada pela primeira vez em 1991. Python é conhecida por sua legibilidade e simplicidade sintática, o que a torna ideal tanto para iniciantes quanto para desenvolvedores experientes.

Características Principais

  • Sintaxe Clara e Legível: Python enfatiza a legibilidade do código, utilizando indentação para delimitar blocos de código, ao invés de chaves ou palavras-chave específicas.
  • Tipagem Dinâmica: As variáveis em Python não precisam ter seu tipo declarado explicitamente, pois o tipo é determinado automaticamente em tempo de execução.
  • Interpretada: Python é executada linha a linha pelo interpretador, o que facilita o desenvolvimento e a depuração.
  • Linguagem Multiparadigma: Python suporta múltiplos paradigmas de programação, incluindo a programação orientada a objetos, imperativa, funcional e procedural.
  • Bibliotecas Padrão Extensas: Python inclui uma ampla gama de bibliotecas padrão, que facilitam a execução de tarefas comuns sem a necessidade de instalar pacotes adicionais.

Estrutura Básica de um Programa em Python

# Este é um comentário em Python
print("Hello, World!")  # Imprime uma mensagem na tela

Tipos de Dados

Python possui diversos tipos de dados integrados que são utilizados para armazenar diferentes tipos de informações. Os principais tipos de dados são:

  • Números: Inteiros (int), números de ponto flutuante (float) e números complexos (complex).
  • Sequências: Strings (str), listas (list), tuplas (tuple).
  • Mapas: Dicionários (dict).
  • Conjuntos: Conjuntos (set), conjuntos imutáveis (frozenset).
  • Booleanos: True e False.

Exemplo de Uso de Tipos de Dados

# Números
a = 5  # int
b = 3.14  # float
c = 1 + 2j  # complex

# Strings
s = "Hello, World!"

# Listas
lista = [1, 2, 3, 4, 5]

# Tuplas
tupla = (1, 2, 3, 4, 5)

# Dicionários
dicionario = {"chave1": "valor1", "chave2": "valor2"}

# Conjuntos
conjunto = {1, 2, 3, 4, 5}

Operadores

Python oferece uma variedade de operadores que permitem realizar operações sobre variáveis e valores. Eles podem ser categorizados em:

    Operadores Aritméticos: Utilizados para operações matemáticas.

    • + (adição): Soma dois valores.
    • - (subtração): Subtrai o segundo valor do primeiro.
    • * (multiplicação): Multiplica dois valores.
    • / (divisão): Divide o primeiro valor pelo segundo.
    • % (módulo): Retorna o resto da divisão do primeiro valor pelo segundo.
    • ** (exponenciação): Eleva o primeiro valor à potência do segundo.
    • // (divisão inteira): Realiza a divisão e retorna apenas a parte inteira do resultado.

    Operadores de Comparação: Utilizados para comparar dois valores.

    • == (igual a): Retorna True se os valores forem iguais.
    • != (diferente de): Retorna True se os valores forem diferentes.
    • > (maior que): Retorna True se o primeiro valor for maior que o segundo.
    • < (menor que): Retorna True se o primeiro valor for menor que o segundo.
    • >= (maior ou igual a): Retorna True se o primeiro valor for maior ou igual ao segundo.
    • <= (menor ou igual a): Retorna True se o primeiro valor for menor ou igual ao segundo.

    Operadores Lógicos: Utilizados para combinar expressões condicionais.

    • and: Retorna True se ambas as expressões forem verdadeiras.
    • or: Retorna True se pelo menos uma das expressões for verdadeira.
    • not: Inverte o valor lógico da expressão.

    Operadores Bit a Bit: Operam em bits individuais de números inteiros

    • & (E bit a bit): Retorna 1 em cada posição de bit onde ambos os bits correspondentes são 1.
    • | (OU bit a bit): Retorna 1 em cada posição de bit onde pelo menos um dos bits correspondentes é 1.
    • ^ (OU Exclusivo bit a bit): Retorna 1 em cada posição de bit onde apenas um dos bits correspondentes é 1.
    • ~ (NOT bit a bit): Inverte todos os bits do operando.
    • << (deslocamento à esquerda): Desloca os bits do operando à esquerda pelo número de posições especificado.
    • >> (deslocamento à direita): Desloca os bits do operando à direita pelo número de posições especificado.

Controle de fluxo

O controle de fluxo em Python permite que você direcione a execução do programa com base em condições e laços. Os principais elementos de controle de fluxo são:

  • Condicionais: if, elif, else são usados para executar diferentes blocos de código com base em condições.
          
x = 10
y = 20

if x > y:
    print("x é maior que y")
elif x == y:
    print("x é igual a y")
else:
    print("x é menor que y")
  • Laços de Repetição: for e while são usados para repetir a execução de um bloco de código.
      
# Laço for
for i in range(5):
    print(i)

# Laço while
contador = 0
while contador < 5:
    print(contador)
    contador += 1
  • Controle de Laços: break e continue são usados para controlar a execução dos laços.
      
# Uso de break
for i in range(10):
    if i == 5:
        break
    print(i)

# Uso de continue
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

Funções

As funções em Python são definidas usando a palavra-chave def e são usadas para encapsular um bloco de código que pode ser reutilizado. Uma função pode receber argumentos e retornar valores.

        def saudacao(nome):
    return f"Olá, {nome}!"

print(saudacao("Mundo"))

Classes e Objetos

Python é uma linguagem orientada a objetos. Classes são definidas usando a palavra-chave class, e objetos são instâncias de classes.

        
class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def saudacao(self):
        return f"Olá, meu nome é {self.nome} e eu tenho {self.idade} anos."

pessoa = Pessoa("João", 30)
print(pessoa.saudacao())

Manipulação de Arquivos

Python oferece funções integradas para ler e escrever arquivos.

        
# Escrita em arquivo
with open('arquivo.txt', 'w') as file:
    file.write("Hello, World!")

# Leitura de arquivo
with open('arquivo.txt', 'r') as file:
    content = file.read()
    print(content)

Bibliotecas Padrão

Python possui uma rica biblioteca padrão que inclui módulos para diversas tarefas, como:

  • os: Interação com o sistema operacional.
  • sys: Parâmetros e funções específicos do sistema.
  • math: Funções matemáticas.
  • datetime: Manipulação de datas e horas.
  • json: Leitura e escrita de JSON.
  • re: Expressões regulares.

Exemplo de uso de Bibliotecas Padrão

import os
import sys
import math
import datetime
import json
import re

# Usando os para listar arquivos em um diretório
print(os.listdir('.'))

# Usando math para calcular a raiz quadrada
print(math.sqrt(16))

# Usando datetime para obter a data e hora atual
print(datetime.datetime.now())

# Usando json para converter um dicionário em uma string JSON
dicionario = {"chave1": "valor1", "chave2": "valor2"}
json_string = json.dumps(dicionario)
print(json_string)

# Usando re para encontrar todas as ocorrências de um padrão em uma string
padrao = r'\d+'
texto = "Existem 2 maçãs e 5 bananas"
print(re.findall(padrao, texto))

Bibliotecas Externas

Além das bibliotecas padrão, Python possui um vasto ecossistema de bibliotecas externas que podem ser instaladas usando o pip. Algumas das mais populares incluem:

  • NumPy: Computação numérica.
  • Pandas: Análise de dados.
  • Matplotlib: Visualização de dados.
  • Requests: Requisições HTTP.
  • Flask/Django: Desenvolvimento web.

Exemplo de Uso de Bibliotecas Externas

# Instalação de bibliotecas externas
# pip install numpy pandas matplotlib requests flask django

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import requests
from flask import Flask

# NumPy: Operações com arrays
array = np.array([1, 2, 3, 4, 5])
print(array * 2)

# Pandas: Manipulação de DataFrames
df = pd.DataFrame({'coluna1': [1, 2, 3], 'coluna2': [4, 5, 6]})
print(df)

# Matplotlib: Criação de gráficos
plt.plot([1, 2, 3], [4, 5, 6])
plt.show()

# Requests: Requisições HTTP
response = requests.get('https://api.github.com')
print(response.json())

# Flask: Criação de uma aplicação web básica
app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, Flask!"

if __name__ == '__main__':
    app.run(debug=True)
